Range Control
  • Common Console Advanced Visualization Engine
  • Cox Range Control Center
  • Ground Truth Database/Database Integration
  • Range Ops Real-Time Display System
  • Transportable Range Augmentation Control System
    • Return to Technology
    The White Sands Test Center operates and manages the Range for test customers. Range personnel schedule tests, control range operations, operate range instrumentation, process collected data, manage the Range communications system, the flight termination transmission systems, and provide frequency surveillance.

    The Cox Range Control Center (RCC) is the operations control, data acquisition and display facility and addresses all mission test control requirements. Ground and aerial target control support is provided, along with weather data, range timing, recovery, and explosive ordnance disposal support.

    The RCC processes and displays data in real-time for project mission support, flight safety, and range control. The range instrumentation acquires and tracks objects in flight to provide data for computing position, velocity, acceleration, attitude, events, trajectory, intercept, miss distance, pitch, yaw, roll, and object internal measurements. Data that is correlated by the timing, is generated and distributed throughout the Range.

    Other facilities within the RCC include the Real Time Data Processing System (RTDPS), the Telemetry Data Center (TDC), and Operations Control and Display Facility. The RTDPS provides real-time data processing for multiple, simultaneous, simple missions or multi-computer support on a single complex mission. The TDC real-time processed data are displayed on strip chart recorders, standard video monitors, and interactive graphics displays. The RCC also includes the Drone Formation Control System, the Drone Target Control System, and the Airspace Display and Control Center.

    Range Support is defined as a customer-tailored unique set of data collection instrumentation for the particular test scenarios. In addition to the direct data collection instrumentation needed to meet the User Project test requirements, certain "derived" sets of data are produced and distributed to other users during the test. The Flight Safety and Range Control functions are critical users of these "derived" sets of data during execution of live tests to ensure successful completion of the exercises.
